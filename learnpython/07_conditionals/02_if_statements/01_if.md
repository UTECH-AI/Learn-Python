# If statements

- An "if statement" is written by using the if keyword.

```py
a = 33
b = 200
if b > a:
  print("b is greater than a")
  ```

  ```py
a = 200
b = 33
c = 500
if a > b and c > a:
  print("Both conditions are True")
  ```

```py
a = 200
b = 33
c = 500
if a > b or a > c:
  print("At least one of the conditions is True")

```
