# Paramaters & Arguments 

- A paramater is a placeholder value 
- An argument is the actual value 


```py
def say_hi(Genna):
    print("Hi "+ Genna )


say_hi("AI CLUB")
```

```py
def say_hi(Genna, age):
    print("Hi "+ Genna + age )


say_hi("AI CLUB ", "14")
```

```py
def say_hi(Genna, age):
    print("Hi "+ Genna + str(age) )


say_hi("AI CLUB ", 14)
```
